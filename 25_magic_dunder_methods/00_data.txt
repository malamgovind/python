Magic (Dunder) Methods એ Python ના Special Double-Underscore Methods છે 
જે Object નું Built-in Operations, Operators અને Runtime Behavior સાથેનું 
Integration અને Customization નિયંત્રિત કરે છે.

* : Python જ્યારે Object પર કોઈ Special Operation કરે છે ત્યારે તે પાછળથી Magic Method Call કરે છે.
