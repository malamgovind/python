Important Assertions

| Assertion        | Use             |
| ---------------- | --------------- |
| assertEqual()    | Equal Values    |
| assertNotEqual() | Not Equal       |
| assertTrue()     | True Condition  |
| assertFalse()    | False Condition |
| assertIn()       | Value Exists    |
| assertIsNone()   | Check None      |
| assertRaises()   | Expected Error  |

Important Classes

| Class          | Use         |
| -------------- | ----------- |
| TestCase       | Create Test |
| TestSuite      | Group Tests |
| TextTestRunner | Run Tests   |
| Mock           | Fake Object |
