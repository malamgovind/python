# Iteration = Loop દ્વારા collection ના elements ને એક પછી એક process કરવાની પ્રક્રિયા.
# Iteration એટલે કોઈ collection (list, tuple, string, set, dictionary) ના elements ને એક પછી એક access કરવી.


-- for
-- while
-- range()
-- enumerate()
-- zip()
-- reversed()
-- sorted()
-- iter()
-- next()  


| Iterable                        | Iterator                 |
| ------------------------------- | ------------------------ |
| list, tuple, string, set, dict  | `iter()` થી બને છે       |
| `for` loop માં સીધું વાપરી શકાય | `next()` થી value મળે    |
| Example: `[1,2,3]`              | Example: `iter([1,2,3])` |
